Przed rozwiązywaniem łamigłówek potrzebujemy zainstalować jedną bibliotekę.

W terminalu wykonujemy polecenie:
gem install rspec

Po instalacji możemy zająć się rozwiązywaniem łamigłówek z pliku puzzles_spec.rb

Żeby sprawdzić poprawność naszych rozwiązań, uruchamiamy polecenie:
rspec puzzles_spec.rb
